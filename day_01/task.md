### 🧠 Task: “Daily Expense Analyzer” (Python + basic data handling)

Create a Python script that:

1. Reads a file named `expenses.txt`
   (you create it manually with sample data like below)

   ```
   Food,250
   Transport,100
   Food,300
   Entertainment,500
   Transport,150
   ```

2. Process the data and:

   * Calculate **total expenses**
   * Calculate **total per category**
   * Find the **highest spending category**

3. Print a clean summary like:

   ```
   Total: 1300
   Food: 550
   Transport: 250
   Entertainment: 500
   Highest: Food
   ```

4. Save the summary into a file called `report.txt`

---

* Use a **dictionary**
* Handle at least one possible error (e.g., bad format)
